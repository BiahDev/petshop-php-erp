## 📋 Sobre o projeto

Banho & Tosa - É um sistema de gestão para um petshop, vai desdo o frente de caixa até os relatorios, cuidando assim de varias etapas e processos de um petshop.

## ✨ Features
- Frente de caixa
- Controle de estoque
- Relatorios com gráficos
